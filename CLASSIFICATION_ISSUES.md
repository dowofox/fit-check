# NAES Classification Issues

## 목적

이 문서는 특정 상품명, 키워드, 소재 표기 또는 쇼핑몰 분류에서 발견한 예외를 모은다.

개별 예외를 발견할 때마다 즉시 분류 엔진을 수정하지 않는다. 반복성과 제품 영향을 확인해 공통 문제로 묶을 수 있을 때만 `ROADMAP.md`의 적절한 핵심 영역으로 승격한다.

## 처리 원칙

- 단일 상품명이나 키워드 오분류는 먼저 이 문서에 기록한다.
- 같은 원인이 여러 상품 또는 쇼핑몰에서 반복되는지 확인한다.
- 상품 하나만 통과시키는 하드코딩보다 재사용 가능한 공식 정보·구조화 데이터·분류 규칙을 우선한다.
- 기존 상품을 새 규칙이 잘못 분류하지 않는 회귀 사례를 함께 정의한다.
- 해결 후에는 항목 상태와 연결된 커밋을 기록한다.

## 즉시 P0 승격 기준

다음 중 하나에 해당할 때만 일반 대기열을 건너뛴다.

- 분류 처리로 앱이 크래시하거나 등록 자체가 불가능하다.
- 기존 옷장 데이터를 잘못 덮어쓰거나 잃을 위험이 있다.
- 잘못된 분류가 계절·핏·코디 추천에 연결되어 사용자에게 명백히 잘못된 착용 또는 구매 판단을 제시한다.

단순 별칭 누락, 드문 상품명, 표시상 세부 분류 차이는 즉시 P0가 아니다.

## 승격 기준

다음 질문에 모두 답할 수 있을 때 제품 작업 후보로 승격한다.

1. 같은 원인이 두 개 이상의 실제 사례에서 반복되는가?
2. 특정 상품명이 아닌 공통 규칙으로 해결할 수 있는가?
3. 옷 등록, 코디 추천 또는 핏·사이즈 설명의 품질에 영향을 주는가?
4. 예상되는 회귀 위험과 검증 사례가 정의되어 있는가?

## 미해결 이슈

현재 등록된 미해결 분류 이슈는 없다.

새 이슈는 아래 형식으로 추가한다.

```md
### CLS-000 · 짧은 제목

- 상태: 관찰 / 반복 확인 / 로드맵 승격 / 해결
- 발견일: YYYY-MM-DD
- 영향 영역: 옷 등록 / 코디 추천 / 핏·사이즈 추천
- 상품 또는 쇼핑몰:
- 현재 결과:
- 기대 결과:
- 반복 사례:
- 사용자 영향:
- 공통 원인 가설:
- 필요한 회귀 테스트:
- 연결 커밋:
```

## 완료 이슈

이 문서 도입 이전의 세부 분류 수정은 Git 이력에 보존되어 있다. 앞으로 새로 접수하는 예외부터 이 문서에서 상태를 관리한다.
