# NAES Roadmap

## 작업 선택 규칙

항상 다음 순서로 작업한다.

1. P0
2. 가장 위 작업
3. 하나만 구현
4. 완료 후 다음 작업

절대로 여러 기능을 한 번에 구현하지 않는다.

새 작업을 시작하기 전에 이미 구현된 기능인지 먼저 확인한다.

## P0

현재 P0는 1차 완료 상태다.

완료된 항목:

- 링크 등록을 기본 등록 방식으로 변경
- 사진 등록 축소
- 추천 화면 단순화
- AI 내부 정보 숨김
- 소재 검증
- 프로필 단순화
- 상황에 따른 추천 코디창을 추가
- 옷장에 옷을 지금보다 세세히 구분할 수 있는 필터를 추가

새 P0 작업을 추가할 때는 이 섹션 맨 위에 하나만 추가한다.

## P1

- 추천 피드백
- 착용 기록
- 옷장 부족 분석
- 데이터 백업

## P2

- 서버 배포
- 로그인
- 동기화

## 완료 기준

- 기존 기능을 삭제하지 않는다.
- 사용자 화면은 결과 중심으로 단순해진다.
- `npm run lint`를 통과한다.
- `npx tsc --noEmit`을 통과한다.
- 로컬 커밋을 만들고 SHA를 보고한다.
- 검증이 모두 통과하면 `origin/main`에 자동 push한다.
- 문제가 있거나 작업 범위가 모호하면 push하지 않고 사용자에게 먼저 묻는다.
